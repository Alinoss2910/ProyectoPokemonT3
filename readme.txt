Como herramienta hemos utilizado Bootstrap, ya que nos parece mucho más fácil realizar un página web con ello.

Y como buenas prácticas hemos utilizado el mismo formato en las páginas junto a la paleta de colores que ya argumentamos en el documento, 
el código no se repite, hemos utilizado imágenes de alta calidad y personalizadas por nosotros, 
nos hemos concentrado en un público como es el mundo del Pokémon, 
hemos comprobado y revisado el código antes de enviarlo y hemos priorizado la legibilidad, 
para que el público que nos lea, sea fácil de entender la página.
